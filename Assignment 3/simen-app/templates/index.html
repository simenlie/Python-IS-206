Signup

Signup lets a user signup with a username, password, and a email(optional). It stores the user data in a database, and after signing up the user gets logged in and a cookie is set for the user
To successfully signup, you will need to follow these rules:
• Username: must contain no spaces and have a minimum length of 4 characters
• Password: Minimum length of 6 characters, validate by retyping the password
• Email: is optional, but if you choose to have one, it needs to be a correct email
[LINK TO SIGNUP]

Login

Login lets a user that already have signed up login. When successfully logged in, a cookie is made for the user, and the user would have to hit logout to login/signup again
To successfully login, you will need to follow these rules:
• Username: must be stored in the database(same username as signing up)
• Password: same as above
